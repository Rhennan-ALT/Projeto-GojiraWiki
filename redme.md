# 🦖 Gojira Wiki

Projeto de wiki interativa focada no universo de monstros gigantes (Kaijus), com destaque para o Godzilla e personagens relacionados.

---

## 🚀 Sobre o Projeto

A **Gojira Wiki** é uma aplicação web em desenvolvimento com foco em:

* Exibir informações sobre personagens (Kaijus)
* Organizar dados como história, poderes e aparições
* Criar uma experiência interativa para navegação entre personagens

---

## 🧱 Estrutura Atual

```
GOJIRA-WIKI/
│
├── assets/
│   └── imgs/          # Imagens dos personagens
│
├── data/
│   └── characters.js  # Dados dos personagens
│
├── pages/             # Páginas secundárias (futuro)
│
├── app.js             # Lógica principal (renderização)
├── index.html         # Página principal
├── style.css          # Estilos adicionais
└── .gitignore         # Arquivos ignorados pelo Git
```
